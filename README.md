# 🏌️ Golf101: The Beginner's Essential Guide

Golf101 is a premium, responsive educational web application designed to help beginners master the fundamentals of golf. From understanding the anatomy of a golf course to selecting the right club for every distance, this guide provides a visually rich and interactive experience.

![Golf101 Preview](https://via.placeholder.com/1200x600/0f1115/ffffff?text=Golf101+Educational+Platform)

## ✨ Key Features

- **Interactive Course Anatomy**: Learn about fairways, roughs, bunkers, and greens with detailed visual guides.
- **Dynamic Distance Chart**: An interactive bar chart showcasing standard carry distances for average players across all clubs.
- **Master the Basics**: Step-by-step instructions on **Grips** (Standard vs. Putter), **Stance** (Top-down view diagrams), and **Swing Mechanics** (Fixing slices and hooks).
- **Club-by-Club Guide**: Detailed mechanical breakdowns of Drivers, Hybrids, Irons, Wedges, and Putters.
- **Essential Gear**: A dedicated section for must-have accessories like gloves, shoes, and divot tools.
- **Premium Design**: A dark-themed, glassmorphic UI built with modern CSS techniques for a state-of-the-art look and feel.
- **Fully Responsive**: Optimized for every device, from high-resolution desktops to mobile phones with tight, content-focused spacing.

## 🚀 Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Properties, Flexbox, Grid, Glassmorphism)
- **Logics**: Vanilla JavaScript (ES6+)
- **Animations**: Intersection Observer API for scroll-triggered fade-ins and dynamic bar loading.
- **Assets**: Custom SVG diagrams and high-quality photography markers.

## 📱 Mobile Optimization

Specifically tailored for mobile users with:
- A sliding glassmorphic **hamburger menu**.
- Tightened white space and optimized typography for smaller viewports.
- Responsive grid layouts that adapt seamlessly from 3 columns to 1.
